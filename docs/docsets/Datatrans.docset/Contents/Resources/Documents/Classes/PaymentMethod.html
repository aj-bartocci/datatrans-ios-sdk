PaymentMethod

@objc(DTPaymentMethod)
@objcMembers
public class PaymentMethod : NSObject, Codable, NSCoding, NSCopying

This is the base class for payment methods. It contains a type to identify the payment method, e.g. Visa or Mastercard.
